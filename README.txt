CYTHON IMPLEMENTATION README

all the material is taken from: 
http://docs.cython.org/en/latest/src/tutorial/cython_tutorial.html
and
http://docs.cython.org/en/latest/src/userguide/numpy_tutorial.html#numpy-tutorial

We are using the setup.py method mentioned in the cython docs. We also assume that cython library is appropirately installed.

run this command in terminal:
python setup.py build_ext --inplace

Now execute the python script, you are done!
